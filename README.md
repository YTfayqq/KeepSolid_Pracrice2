# KeepSolid_Pracrice2
Опис

BookStream — це веб-додаток для читання книг. Він дозволяє:

Шукати книги за назвою або автором.
Додавати до обраного.
Перемикати темну/світлу тему.
Переглядати деталі книг.

Проект створено для демонстрації роботи з JavaScript, HTML та CSS у рамках практики в KeepSolid. Основні файли знаходяться в папці BookStream/.

Установка

Клонуйте репозиторій:git clone https://github.com/YTfayqq/KeepSolid_Pracrice2.git
Перейдіть до директорії:cd KeepSolid_Pracrice2/BookStream
Відкрийте index.html у браузері.

Використання

Перейдіть на головну сторінку: index.html.
Використовуйте пошук для знаходження книг.
Додайте книгу до обраного, натиснувши на кнопку.
Увійдіть або зареєструйтеся для збереження обраного.

Структура проекту

BookStream/: Основна папка з файлами проекту.
index.html: Головна сторінка.
book.html: Сторінка деталей книги.
login.html: Сторінка входу.
profile.html: Сторінка профілю.
register.html: Сторінка реєстрації.
script.js: JavaScript-логіка (авторизація, тема, обране, пошук).
styles.css: Стилі CSS (теми, адаптивність).
